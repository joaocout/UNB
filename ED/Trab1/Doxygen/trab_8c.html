Detailed Description

Arquivo raiz do trabalho.

É responsável por toda a parte gráfica do programa, como printar menus, printar resultados e solicitar a emtrada de dados ao user, além de ser responsável por chamar os dois modos possiveis: o modo calculadora e o modo resolução de expressões.

◆ calculadora()

void calculadora ( )

Inicia a execução do modo calculadora.

Cria uma pilha e aguarda a entrada de dados via 'scanf';
Caso seja um operando o mesmo é empilhado;
Caso seja um operador, os operandos são desempilhados, a operação realizada, e o resultado empilhado novamente;
Quando o usuário digita 'sair', a funcão retorna ao menu principal.

◆ expressao()

void expressao ( )

Inicia a execução do modo resolução de expressão.

O programa aguarda a entrada de uma expressão na forma infixa via 'scanf';
Primeiramente, cria-se uma pilha para a validação da expressão, sendo que essa pilha é removida ao final do processo;
Caso a expressão seja válida, inicia-se a geração da expressão posfixa, utilizando uma pilha para armazenar os operadores;
Ao final, é exibido se a expressão é válida, sua forma posfixa(caso possível), e seu resultado(caso possível);
Caso o usuário entre com 'sair', a funcão retorna ao menu principal.

◆ menu()

void menu ( )

Mostra o menu principal.

São exibidos os modos de operação ao usuário, e aguarda-se a entrada de dados;
'1' para o modo calculadora;
'2' para o modo resolução de expressões;
'3' para encerrar a execução do programa.
